Remote Mouse

Remote Mouse transforms your iPhone into a wireless trackpad and keyboard for your Mac.

Control the cursor, click, scroll, and type directly from your phone using a fast local network connection.

Features

  • Smooth cursor movement
  • Tap to click
  • Two-finger scrolling
  • Live keyboard typing
  • Fast local network connection
  • Minimal and clean interface

How It Works

Launch the Mac companion app, connect your iPhone to the same Wi-Fi or hotspot, enter your Mac IP address, and start controlling your Mac instantly.

Built With

  • SwiftUI
  • Network Framework
  • CoreGraphics
  • GitHub Pages
