Remote Mouse Support

If you experience connection issues or have questions about the app, please contact support.

Requirements

  • iPhone and Mac must be connected to the same local network
  • The Mac companion app must be running
  • Accessibility permissions must be enabled on macOS

Common Fixes

  • Restart the Mac companion app
  • Reconnect both devices to the same Wi-Fi or hotspot
  • Verify the correct IP address is entered
  • Allow Local Network access on iPhone
